## 🛠 טכנולוגיות

- **React 18** + **Vite 5**
- **PWA** — manifest + Service Worker (vite-plugin-pwa)
- **CSS** — glassmorphism, 3D flip animation, RTL מלא
- **localStorage** — שמירת שם, התקדמות, streak
- אפס תלויות UI חיצוניות

## 🚀 הרצה מקומית

```bash
# Clone
git clone https://github.com/hilaln2210/arabic-learning
cd arabic-learning

# התקנה
npm install
pip install edge-tts  # קולות ערביים Neural של Microsoft

# הפעלה (שני טרמינלים):
python3 tts_server.py   # טרמינל 1 — שרת הקראה קולית
npm run dev             # טרמינל 2 — אפליקציה

# בניה לפרודקשן
npm run build
```

## 🧠 שיטת הלימוד

כל מילה מגיעה עם **אסוציאציה בעברית** שמקשרת את הצליל לזיכרון קיים:

```
جيش  →  "ג'ייש"   →  🧠 ג'יפ של הצבא
سلام  →  "סלאם"   →  🧠 שלום = סלאם — כמעט זהה!
خمسة  →  "ח'מסה"  →  🧠 חמסה = יד = 5 אצבעות!
عين   →  "עין"    →  🧠 עין = עין — זהה ממש!
يد    →  "יד"     →  🧠 יד = יד — זהה ממש!
```

המוח זוכר פי **5 מהר יותר** עם אסוציאציות ויזואליות.
